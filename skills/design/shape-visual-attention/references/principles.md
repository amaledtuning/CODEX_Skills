# Visual Attention Principles

Source anchor: paraphrased from the design book in `D:\Project Skills\Книги по дизайну\100_glavnyh_principov_dizayna_kak_uderzhat_vnimanie.pdf`, especially perception, peripheral vision, recognizable forms, affordance, and color sections around pp. 14-38.

## Core Principles

- Design for perceived meaning, not raw layout. People notice what contrast, grouping, scale, and position make salient.
- Make the primary message or action win the first glance. A visual with several equal focal points usually communicates none of them clearly.
- Use peripheral vision intentionally. Critical alerts, status changes, and action cues should remain detectable without tiny-detail focus.
- Prefer recognizable forms and familiar iconography. Novel visuals can be beautiful but should not make common actions harder to identify.
- Make affordances visible. Buttons, controls, links, and interactive objects should look actionable before the user reads an explanation.
- Use grouping to create meaning. Proximity, shared backgrounds, alignment, and repetition tell users what belongs together.
- Treat color as assistive, not primary evidence. Pair color with labels, position, shape, pattern, icon, or text.
- Check cultural and accessibility risk in color meaning. Avoid relying on red/green or harsh red/blue pairings as the sole distinction.

## Practical Checklist

- Name the one thing that should be noticed first.
- Reduce competing accents near the primary focus.
- Increase hierarchy through scale, weight, contrast, whitespace, or position before adding decoration.
- Group related items and separate unrelated items with space or containers.
- Verify every state cue has at least two signals: for example color plus icon, label, position, or shape.
- Make the next action visually obvious without a paragraph of explanation.
- For slides and PDFs, ensure the page can be understood in a few seconds.
- For images and posters, ensure the subject, message, and call to action do not compete.

## Anti-Patterns

- Several CTAs with equal visual weight.
- Decorative elements that overpower the message.
- Important warnings represented only by color.
- Icons that require guessing because they are unfamiliar or unlabeled.
- Dense grids where grouping is only implied by data order.
- Low-contrast secondary text that carries essential meaning.

## Output Pattern

When using this skill, produce:

1. Primary attention target.
2. Current attention blockers.
3. Specific hierarchy/composition changes.
4. Redundant non-color cues for important states.
5. Format-specific notes for UI, slide, PDF, image, poster, or infographic.
